# User Stories

| As a       | I want to                        | So that I can                                    |
| ---------- | -------------------------------- | ------------------------------------------------ |
| visitor    | view all Pokémon                 | see the list of existing Pokémon                 |
| visitor    | search for a Pokémon             | check its specifications                         |
| visitor    | view all types                   | see the list of existing types                   |
| visitor    | search for a type                | filter Pokémon by type                           |
| visitor    | view all teams                   | see the list of existing teams                   |
| visitor    | search for a team                | see the Pokémon on that team                     |
| visitor    | access a registration form       | create an account to build and save my teams     |
| player     | create a team                    | have my own team                                 |
| player     | rename a team                    | keep the team and only change its name           |
| player     | add a Pokémon to a team          | update my team                                   |
| player     | remove a Pokémon from a team     | update my team                                   |
| player     | delete a team                    | remove team I no longer want                     |
| player     | access a login form              | log in and manage my teams                       |
